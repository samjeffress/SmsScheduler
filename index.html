SMS Scheduler

Trickle a bunch of messages to be sent out over time.

Download this project as a .zip file Download this project as a tar.gz file

What?

  • Trickle a number of SMS over a time period or with a defined time gap between the messages
  • Scheduling of individual SMS
  • Pausing / rescheduling of SMS
  • Directly sending SMS

Why?

If you need to contact your customers to help them catch up on payments or recommit, but your contact centre can't handle everyone calling at once the SMS Scheduler is the ideal tool to help you manage the flow.

Who?

If you're in the non-profit space we'd love to be able to help you out!

Other features

  • Easily send individual SMS messages using the API or NServiceBus messages
  • Integrate with an existing solution to add SMS functionality using the API or NServiceBus messages
  • Subscribe to events to save a sent message into your CRM for tracking and get a more comprehensive view of your customers.

Built for Windows using NServiceBus and RavenDB, powered by Twilio.
